## VPS选购常见问题解答（FAQ）

### Q1：免备案VPS和需要备案的有什么区别？

**A：** 免备案VPS通常指海外节点（香港、美国、日本等），购买后可直接使用，无需进行ICP备案，适合外贸、跨境业务。需要备案的主要是国内节点，必须完成实名认证和ICP备案才能用于网站访问，但访问速度和稳定性更好。

### Q2：CN2 GIA线路有什么优势？值得买吗？

**A：** CN2 GIA是中国电信的高端回国线路，三网直连，延迟低、丢包率低，特别适合需要回国访问的业务。如果你主要面向国内用户，CN2 GIA线路能显著提升访问体验，值得投资。

### Q3：高防VPS能防御多大攻击？怎么选？

**A：** 高防能力从20G到T级不等。普通网站20-50G足够；游戏、金融等业务建议100G以上；大型平台需要T级防护。阿里云、腾讯云、天翼云提供T级清洗，衡天云、恒创云等提供100-300G高防。

### Q4：按小时计费的VPS适合什么场景？

**A：** 适合临时测试、短期项目、开发调试、学习实验等场景。Vultr、LightNode、AkileCloud支持按小时计费，用完即删，成本可控。

### Q5：住宅IP和双ISP的VPS有什么用？

**A：** 主要用于解锁流媒体（Netflix、Disney+等）、社交媒体营销、跨境电商账号运营等。荫云、丽萨主机、AkileCloud提供此类IP，不易被识别为机房IP。

### Q6：1美元左右的VPS能用吗？有什么坑？

**A：** RackNerd、搬瓦工等提供的低价VPS适合个人学习、测试、小流量网站。注意：性能有限、带宽较小、无高防能力，不适合商业项目。购买前确认是否支持支付宝、是否有隐藏费用。

### Q7：企业用户应该如何选择VPS？

**A：** 建议选择阿里云、腾讯云、京东云、天翼云、UCloud等一线云厂商。优势：提供正式合同和发票、SLA保障、备案支持、企业级安全防护、技术支持响应快。虽然价格较高，但合规性和稳定性更有保障。

## VPS选购注意事项

1. **线路测试**：购买前先用商家提供的测试IP测试延迟和丢包率
2. **退款政策**：确认是否支持无理由退款及退款期限
3. **带宽限制**：注意是独享带宽还是共享带宽，是否有流量限制
4. **IP更换**：了解IP更换政策和费用，特别是做外贸的用户
5. **技术支持**：确认客服响应时间和支持渠道（工单、QQ、微信等）
6. **付款方式**：确保支持你方便的支付方式（支付宝、微信等）
7. **使用条款**：阅读TOS，确认是否允许你的业务类型
